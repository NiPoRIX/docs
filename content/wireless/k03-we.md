**K:03 – Wireless Edition** — беспроводная, эргономичная, механическая сплит-клавиатура с хотсвап-системой для быстрой замены MX-совместимых свитчей и возможностью установить до 3 хотсвап энкодеров на каждую половинку

## Обзор

![image](/images/k03_we/k03_we_ru.png)

1. кнопка: однократное нажатие — reset, двукратное нажатие (быстро) — bootloade
2. интегрированное кольцо MagSafe для УСН
3. силиконовые ножки
4. вспомогательное окно для разбора корпуса клавиатуры
5. переключатель питания клавиатуры
6. прозрачная панель
7. место для установки MX-свитча или хотсвап-энкодера
8. разъем USB-C

## Подключение

- K:03 – Wireless Edition работает с устройствами под управлением Windows, macOS, Linux, Android, iOS
- Подключение клавиатуры осуществляется посредством Plug and Play (PnP), дополнительная установка драйверов не требуется
- Клавиатура поставляется готовым для подключения к вашему устройству с актуальной на момент покупки прошивкой

{{% details title="Проводное подключение" closed="true" %}}

Клавиатура поддерживает проводное подключение через разъем USB-C. Используйте комплектный кабель для подключения, либо любой другой кабель с USB-C разъемом и поддержкой передачи данных.
{{< callout type="info" >}}
  **Левая половинка** K:03 – Wireless Edition является **главной (master)** и предназначена для проводного подключения
{{< /callout >}}
{{% /details %}}

{{% details title="Подключение напрямую (без Ergohaven's Qube)" closed="true" %}}
1. **Включите питание на левой и правой половинках клавиатуры** - половинки уже синхронизированы между собой
2. **Выберите профиль подключения на клавиатуре** - если нужно подключить на несколько устройств, для первого подключения можете пропустить этот шаг
3. **Подключите клавиатуру к устройству** - включите Bluetooth на устройстве, затем выполните сопряжение в настройках устройства
{{% /details %}}


{{% details title="Подключение через Ergohaven's Qube" closed="true" %}}
**Ergohaven's Qube** — универсальный донгл для беспроводных клавиатур Ergohaven, который отображает полезную информацию и снижает энергопотребление клавиатуры

1.  **Используя Type-C кабель подключите Ergohaven's Qube к компьютеру** и дождитесь загрузки Qube
2.  **Включите питание на левой половинке клавиатуры** - половинка автоматически подключится к Qube
3.  **Включите питание на правой половинке** - половинка также подключится к Qube, завершая настройку
{{% /details %}}

{{% details title="Подключение дополнительных устройств" closed="true" %}}

1. Включите Bluetooth на устройстве, к которому подключаете клавиатуру
2. Включите клавиатуру, переведя переключатели питания половинок на нижних крышках в положение *ВКЛ*, клавиатура перейдёт в режим сопряжения
3. Переключите BT‑профиль на клавиатуре при помощи соответствующих кнопок (3 слой в раскладке по умолчанию)
3. Начните поиск Bluetooth на устройстве и найдите клавиатуру в списке
4. Подключите клавиатуру к устройству
{{< callout type="info" >}}
   [Добавление беспроводного подключения к устройствам через Qube](https://docs.ergohaven.xyz/wireless/qube/)
{{< /callout >}}

## Прошивка

Прошивка требуется в случаях, когда:
- Вышло обновление прошивки на [keymap_hub](https://github.com/ergohaven/keymap_hub)
- Требуется вернуть клавиатуру к заводским настройкам
- Вы используете собственные настройки в прошивке через [форк](https://docs.ergohaven.xyz/zmk/fork-sync/)

{{% steps %}}

### Подготовка к прошивке

Скачайте актуальные файлы прошивок *settings_reset* и *K:03 v4* для левой и правой половинок с [keymap_hub](https://github.com/ergohaven/keymap_hub)

### Прошивка для сброса настроек

Подключите USB-C кабелем устройство с прошивкой (ПК, ноутбук и т.п.) и левую половинку клавиатуры, дважды нажмите кнопку *Reset* с обратной стороны клавиатуры

{{< callout type="info" >}}
  Клавиатура перейдёт в режим загрузчика и откроется папка-хранилище (если на вашем устройстве отключен автозапуск подключаемых дисков, то перейдите в папку-хранилище вручную)
{{< /callout >}}

Скопируйте или перетащите Uf2 файл *settings_reset*, после чего клавиатура должна перезагрузиться

{{< callout type="info" >}}
  Settings_reset прошивка нужна для затирания внутренней памяти клавиатуры. Она сбрасывает настройки профилей подключения и синхронизации половинок
{{< /callout >}}

{{< callout type="important" >}}
  Повторите те же шаги для правой половинки
{{< /callout >}}


### Завершение прошивки 

Снова дважды нажмите кнопку *Reset* с обратной стороны левой половинки и дождитесь, когда она перейдёт в режим загрузчика (откроется папка-хранилище)

Скопируйте или перетащите Uf2 файл прошивки *K:03 v4* для левой половинки, после чего она должна перезагрузиться
{{< callout type="info" >}}
  Если при копировани прошивки появляется окно с ошибкой копирования, то просто игнорируйте это сообщение
{{< /callout >}}

{{< callout type="important" >}}
  Повторите те же шаги для **правой** половинки, но используйте Uf2 файл прошивки *K:03 v4* уже для **правой** половинки 
{{< /callout >}}

{{% /steps %}}

Прошивка завершена, можно подключать клавиатуру к вашему устройству!
