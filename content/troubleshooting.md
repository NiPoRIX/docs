## Первичная диагностика 

{{< callout type="important" >}} 
Первичная диагностика рекомендуется для исключения типовых проблем не связанных с электронной частью продукции Ergohaven
{{< /callout >}}

{{% steps %}}

### Смена устройства

Проверьте, проявляется ли неисправность на другом устройстве (ПК, ноутбук, планшет, смартфон и т.д). Если используется USB Hub, то попробуйте подключиться напрямую к устройству либо сменить порт USB

### Смена операционной системы

Проверьте, проявляется ли неисправность на другой операционной системе (Windows, macOS, Linux и другие).

### Смена интерфейса подключения

Если используеттся проводное подключение между половинками или компьютером, то замените кабель между половинками, компьютером.

### Возврат к заводским настройкам/обновление прошивки

Перепрошейте на стандартную прошивку ваше устройство от Ergohaven.

### Обновление драйверов

Переустановите драйвер USB или Bluetooth на вашем устройстве (ПК, ноутбук и т.д).

{{< callout type="info" >}}
  Встроенные драйвера Windows не всегда корректно работают на устройстах USB и Bluetooth, используйте драйвера от производителя устройства
{{< /callout >}}

{{% /steps %}}

## Неполадки при нажатии клавиш

{{< details title="Залипание или несрабатывание отдельных клавиш" closed="true" >}}

Поменяйте местами свитчи с дефектного места и исправного, тем самым вы исключите сам свитч. Если после смены свитча проблема сохраняется, а сам свитч исправен на другом месте, то необходима диагностика самой платы устройства. 

{{< /details >}}

## Неполадки трекболов

{{< details title="Плохая точность перемещения указателя, рывки при перемещении" closed="true" >}}

{{< callout type="info" >}}

**Повышенная точность указателя** (также известна как акселерация мыши) — это функция, которая изменяет скорость перемещения курсора в зависимости от скорости движения мыши. При использовании трекбола её часто, чтобы получить предсказуемое линейное перемещение
  
{{< /callout >}}

{{< details title="Windows 10 и Windows 11" closed="true" >}}

В Windows эта функция называется **«Включить повышенную точность указателя»**. Чтобы её отключить:

1. Откройте **Параметры** (нажмите клавиши `Win + I`).
2. Перейдите в раздел **Bluetooth и устройства** → **Мышь**.
3. В правой части окна нажмите на пункт **Дополнительные параметры мыши**.
4. В открывшемся окне «Свойства мыши» перейдите на вкладку **Параметры указателя**.
5. В разделе **Перемещение** снимите галочку с пункта **«Включить повышенную точность указателя»**.
6. Нажмите кнопку **Применить**, затем **ОК**.

> **Примечание:** Изменения вступают в силу сразу. При необходимости вы можете отрегулировать ползунок **Скорость движения указателя** там же.

{{< /details >}}

{{< details title="macOS" closed="true" >}}

Способ отключения зависит от версии операционной системы.

### Для macOS Sonoma и новее:
1. Откройте меню Apple () → **Системные настройки**.
2. В боковом меню выберите **Мышь**.
3. Нажмите кнопку **Дополнительно**.
4. Переключите опцию **Акселерация указателя** в положение **Выкл.**

### Для старых версий macOS:
1. Откройте программу **Терминал** (Finder: `Программы` → `Утилиты`).
2. Введите команду:
   ```bash
   defaults write .GlobalPreferences com.apple.mouse.scaling -1
   ```
и нажмите Enter.
3. Выйдите из системы или перезагрузите компьютер для применения изменений.   
{{< /details >}}

{{< details title="Linux" closed="true" >}}

### Для GNOME (Ubuntu, Fedora и др.):
1. Откройте **Терминал**.
2. Введите команду для отключения акселерации:
   ```
   bash
   gsettings set org.gnome.desktop.peripherals.mouse accel-profile flat
   ```
Для проверки выполните:
```
bash
gsettings get org.gnome.desktop.peripherals.mouse accel-profile
```
В ответ должно появиться значение flat.

Для KDE Plasma:
Откройте **Системные настройки** → **Устройства ввода** → **Мышь**

Найдите пункт **Профиль акселерации** и выберите значение **Плоский**.
